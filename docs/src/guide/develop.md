# TinyAgent 开发指引

## 本地开发

```sh
# 下载代码到本地
$ git clone git@github.com:opentiny/tiny-agent.git

# 全局安装 pnpm
$ npm install pnpm -g

# 安装所需的依赖
$ cd tiny-agent/ && pnpm install

# 启动
$ pnpm dev
```

打开浏览器访问： `http://localhost:5173/`

### 环境配置

复制 `demo-server/.env-example` 内容到 `demo-server/.env` 中，填写自己的api key
